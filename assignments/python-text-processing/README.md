# 📘 Assignment: Python Text Processing

## 🎯 Objective

Master the fundamentals of text processing in Python by working with strings, file I/O operations, and text manipulation techniques. You'll learn to read files, parse text data, clean and transform strings, and write processed results back to files.

## 📝 Tasks

### 🛠️ Build a Text Processing Utility

#### Description

Create a Python program that processes text files to clean, analyze, and transform text data. Your utility should read input files, apply various text transformations, and generate formatted output. The program should handle edge cases gracefully and provide useful statistics about the processed text.

#### Requirements

Completed program should:

- Read text files using file I/O operations (open, read, close)
- Convert text to lowercase and remove leading/trailing whitespace
- Remove punctuation and special characters where appropriate
- Count word frequency and identify the most common words
- Split text into sentences and paragraphs
- Write processed results to output files with proper formatting
- Handle file not found and other I/O exceptions gracefully
- Provide statistics such as word count, character count, and unique words
