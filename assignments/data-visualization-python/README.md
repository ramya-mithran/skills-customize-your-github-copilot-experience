# 📘 Assignment: Python Data Visualization

## 🎯 Objective

Learn to create professional data visualizations using Python libraries like matplotlib and plotly. You'll master chart types, customization techniques, and best practices for presenting data effectively to different audiences.

## 📝 Tasks

### 🛠️ Create Interactive Data Visualizations

#### Description

Build a collection of functions that generate various types of visualizations from different datasets. Your program should create line charts, bar charts, scatter plots, histograms, and pie charts with proper formatting, labels, and styling. Include both static matplotlib visualizations and interactive plotly visualizations.

#### Requirements

Completed program should:

- Create line charts to show trends over time
- Build bar charts for categorical data comparison
- Generate scatter plots to show relationships between variables
- Create histograms to visualize data distributions
- Design pie charts for showing proportions and percentages
- Add titles, axis labels, and legends to all charts
- Apply color schemes and styling for visual appeal
- Include gridlines, annotations, and figure sizing
- Save visualizations to files (PNG, SVG formats)
- Create interactive plots using plotly with hover information
- Handle missing or invalid data gracefully
- Format axes with appropriate scales and tick marks
