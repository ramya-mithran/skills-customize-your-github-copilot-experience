# 📘 Assignment: Data Structures in Python

## 🎯 Objective

Develop proficiency with Python's fundamental data structures: lists, dictionaries, sets, and tuples. You'll learn when and how to use each structure, perform common operations, and apply them to solve real-world problems.

## 📝 Tasks

### 🛠️ Master Python Data Structures

#### Description

Create a collection of functions that demonstrate proficiency with Python's core data structures. Each function should solve a specific problem using the most appropriate data structure, including list manipulation, dictionary operations, set mathematics, and tuple handling.

#### Requirements

Completed program should:

- Create and manipulate lists (append, insert, remove, sort, slice)
- Use comprehensions to build and filter lists and dictionaries
- Work with dictionaries for key-value storage and nested data
- Use sets for unique values and set operations (union, intersection, difference)
- Handle tuples as immutable collections and use them as dictionary keys
- Sort dictionaries by keys and values
- Merge and combine multiple data structures
- Remove duplicates from lists while preserving order
- Find common elements between collections using set operations
- Filter and transform collections using built-in functions (map, filter) or comprehensions
