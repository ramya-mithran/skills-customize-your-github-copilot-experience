# 📘 Assignment: Statistics with Python

## 🎯 Objective

Master statistical analysis and data manipulation using Python's powerful pandas and numpy libraries. You'll learn to load datasets, compute descriptive statistics, perform data cleaning, and extract meaningful insights from raw data.

## 📝 Tasks

### 🛠️ Perform Statistical Analysis on Datasets

#### Description

Create a comprehensive statistical analysis program that loads datasets, performs exploratory data analysis (EDA), calculates descriptive statistics, and identifies patterns and relationships in data. Your program should handle real-world messy data, compute correlations, and generate summary statistics.

#### Requirements

Completed program should:

- Load data from CSV files using pandas
- Display basic information about datasets (shape, columns, data types)
- Compute descriptive statistics (mean, median, mode, std deviation, quartiles)
- Handle missing values appropriately (detect, fill, or remove)
- Calculate correlations between variables and identify relationships
- Group data by categories and compute aggregate statistics
- Identify outliers using statistical methods (IQR, Z-score)
- Create frequency distributions and value counts
- Sort and filter data based on conditions
- Merge and combine multiple datasets
- Compute percentiles and quantiles
- Generate summary reports with key findings
